# Project: Orquestba Landing Page

## Overview

This project is a high-conversion landing page for Orquestba, a company focused on building systems that improve organization and decision-making for businesses.

The goal is not just visual quality, but clarity, performance, and conversion.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Principles

- Keep components simple and reusable
- Prefer server components when possible
- Avoid unnecessary client-side JavaScript
- Optimize for performance and SEO
- All components must be responsive (Mobile - Tablet - Desktop)

## Code Style

- Use clean and minimal abstractions
- Avoid overengineering
- Prefer composition over complexity
- Keep file structure organized by feature/section

## UI/UX Guidelines

- Focus on clarity over decoration
- Each section should have a clear purpose
- Strong hierarchy (title → subtitle → CTA)
- Avoid generic marketing language

## Sections Expected

- Hero
- Services
- About
- Process
- Contact / CTA

## Behavior Expectations

- Always explain decisions briefly when generating code
- Prefer production-ready solutions over examples
- Avoid placeholder text unless explicitly requested

## Design Constraints

- Prioritize clarity and conversion over artistic expression
- Avoid overly experimental layouts
- Keep UI intuitive and familiar to users
- Use strong hierarchy and clear messaging
- Design should feel modern and premium, not experimental

## Goal

Help build a fast, clean, and high-converting landing page.
